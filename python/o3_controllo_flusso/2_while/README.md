# While

Si usa quando si è incerti di quanto potrebbe durare il ciclo. Controlla ad ogni iterazione se la condizione per continuare è ancora valida.

```python
condizione = True 

while condizione:
    pass # fai qualcosa, modifica la condizione all'occorrenza
```
