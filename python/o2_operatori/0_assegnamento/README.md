# Assegnamento

L'operatore di assegnamento associa una variabile (vecchia o nuova) ad un oggetto. Per esempio, creiamo la variabile `a`:

```python
a = 1 + 1 
```
La variabile `a` contiene l'intero `2`. Possiamo far puntare un'altra variabile allo stesso oggetto, cos√¨:

