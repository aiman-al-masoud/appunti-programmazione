# Stringa

Una stringa è una sequenza **immutabile** di char (che sono stringhe di lunghezza uno).

```python
s = "ciao mondo"
# oppure:
s = 'ciao mondo' 
```

Apici singoli o doppi: è esattamente la stessa cosa.
